# 🧮 Современный веб-калькулятор (Glassmorphism Calculator)

![версия](https://img.shields.io/badge/version-1.0.0-blue)
![лицензия](https://img.shields.io/badge/license-MIT-green)

Красивый и функциональный калькулятор с дизайном в стиле **Glassmorphism**. Поддерживает клавиатуру, анимации нажатий, обработку ошибок и модульные тесты.

## 📋 Функциональность

- Базовые арифметические операции: `+`, `-`, `×`, `÷`
- Проценты (`%`), смена знака (`+/-`), десятичная точка (`.`)
- Полная поддержка клавиатуры:
  - Цифры `0-9`
  - Операторы `+`, `-`, `*`, `/`
  - `Enter` или `=` — вычисление
  - `Escape`, `Delete`, `Backspace` — очистка
- Индикация текущей операции (подсветка активного оператора)
- Анимация нажатия и вибрация на мобильных устройствах (опционально)
- Обработка ошибок: деление на ноль → `Error`, автоматический сброс через 1.5 сек
- История последней введённой операции (например, `5 +`)

## 🛠 Технологии

- **HTML5** – структура
- **CSS3** – Glassmorphism, Flex/Grid, адаптивность, анимации
- **JavaScript (ES6+)** – логика, обработка событий, управление состоянием
- **Jest** – модульное тестирование (21 тест)

## 🧪 Запуск и тестирование

### Локальный запуск (без сервера)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/onlyKamilka/calculator2.git
   cd calculator2
   Откройте файл index.html в любом современном браузере.
Запуск тестов

Тесты используют Jest и предназначены для проверки логики (функции calculate и formatNumber). Для их выполнения:

Установите Node.js (если ещё не установлен).
Установите зависимости (Jest):

bash
npm install --save-dev jest
Запустите тесты:

bash
npx jest calculator.test.js
Примечание: файл calculator.test.js импортирует функции calculate и formatNumber из script.js. Для корректной работы тестов необходимо добавить в конец script.js строки:

js
if (typeof module !== 'undefined' && module.exports) {
  module.exports = { calculate, formatNumber };
}
Или использовать модульную систему (например, export), если проект будет переведён на модули.
